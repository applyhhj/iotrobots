vDelay


Requires:
Java on both systems
A ntpclient to set the system time.
	http://www.meinberg.de/english/sw/ntp.htm  For windows systems
	Mac OS, Linux and unix come with ntpclient.
Works on all OSs.



How to run:
Run the renderer.bat on the sender system.
Point the webcam to the barcode.
Run reader.bat on the receiver, follow the screen commands.


Key Points
On the receiver side disable Aero.
From webcam settings decrease shutter speed (1ms is good) and increase gain.
